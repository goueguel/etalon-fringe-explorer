# Changelog

All notable changes to this project will be documented in this file.

## v0.1.1 - 2026-06-30

### Added

- Standalone educational web app for exploring Airy transmission and etalon fringes.
- Interactive sliders for refractive index, gap length, internal angle, and reflectivity.
- Derived readouts for FSR, finesse, peak FWHM, fringe amplitude, fringe order, fringes in view, and fixed-wavenumber transmission.
- Presets for uncoated windows, AR coatings, high-finesse cavities, oblique incidence, and reset.
- Drift simulation showing detector signal oscillation at fixed wavenumber as refractive index changes over time.
- GitHub-ready repository metadata, documentation, and GitHub Pages deployment workflow.
